## 计算机图形学的方向
计算机图形学可以笼统的分为三个大方向

1. rendering（渲染）：光栅图形显示
2. geometry（几何）：几何造型技术
3. simulation（仿真、模拟）：真实感图形显示

除掉三个大方向以外还有

1. 3D printing（3D打印）,
2. visualization（数据可视化）
3. GPGPU（通用GPU编程）
4. 人机交互
5. 等等……
