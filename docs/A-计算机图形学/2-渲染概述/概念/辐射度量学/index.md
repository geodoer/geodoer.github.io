---
title: 辐射度量学
---

辐射度量学是一种用来度量电磁场辐射（包括可见光）的手段。

| 物理量 | 说明 | 单位 |
| - | - | - |
|  [辐射通量](./../辐射通量) | 光源所输出的总能量 | 瓦特 |
| [立体角](./../立体角) | 单位球体上的一块面积 | |
| [辐射强度](./../辐射强度) | 光源（即一束光），打在单位立体角上的能量 | |
| [辐射率](./../辐射率) | 光源（即一束光），打在单位立体角、单位面积上的能量  | |
| [辐射照度](./../辐射照度) | 所有光线，打在单位立体角、单位面积上的能量 | |
