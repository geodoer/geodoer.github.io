## linux编译
以ubuntu为例。

**一、安装系统库**
```powershell
sudo apt-get install libassimp-dev
sudo apt-get install libassipm3
sudo apt-egt install assimp-utils
```

**二、编译**
找到release版本：[https://github.com/assimp/assimp/releases/tag/v5.1.4](https://github.com/assimp/assimp/releases/tag/v5.1.4)
```powershell
sudo apt-get install cmake

wget https://github.com/assimp/assimp/archive/refs/tags/v5.1.4.tar.gz
tar -zxvf v5.1.4.tar.gz

cd assimp-5.1.4/
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/home/cm/codes/Tools/3rdlib_linux/assimp ..

make 
sudo make install
```

使用：
```powershell
g++ *** -lassimp
```
