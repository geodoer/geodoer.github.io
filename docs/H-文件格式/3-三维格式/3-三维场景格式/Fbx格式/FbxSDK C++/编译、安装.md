## Linux
直接下载Linux版本：[https://www.autodesk.com/content/dam/autodesk/www/adn/fbx/2020-0-1/fbx202001_fbxsdk_linux.tar.gz](https://www.autodesk.com/content/dam/autodesk/www/adn/fbx/2020-0-1/fbx202001_fbxsdk_linux.tar.gz)

1.  解压文件
```
tar -zxvf fbx202001_fbxsdk_linux.tar.gz
```

2.  请确保您对安装具有执行权限程序(FBX202001_FBXFILESDK_LINUX):
```
% ./FBX202001_FBXFILESDK_LINUX /usr
```

3.  执行安装， usr代表安装目录
```
% ./FBX202001_FBXFILESDK_LINUX /usr
```
