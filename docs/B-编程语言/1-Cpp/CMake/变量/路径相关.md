| 变量名                                   | 说明                                     | 示例                    |
| ---------------------------------------- | ---------------------------------------- | ----------------------- |
| `CMAKE_SOURCE_DIR`或`PROJECT_SOURCE_DIR` | 工程根目录（顶部CMakeLists.txt所在目录） | `.\`                    |
| `CMAKE_BINARY_DIR`或`PROJECT_BINARY_DIR` | 工程build目录 | `.\build` |
| `CMAKE_CURRENT_LIST_DIR`                 | 当前`CMakeLists.txt`或`*.cmake`所在目录               | `.\submodules\HiOpenGL` |
