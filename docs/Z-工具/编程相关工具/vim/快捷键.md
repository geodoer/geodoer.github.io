---
title: VIM快捷键
---

## 选中

| 快捷键 | 说明 |
| -|-|
|`ve` | `v`可视化；`e`选中一个单词|

## 编辑
| 快捷键|说明|
|- |-|
| `shift+v`, 选择多行, `>`或`<` | 对多行进行Table缩进 |

## 移动
| 快捷键 | 说明|
| -  |- |
| | **行内,单词** |
| `w` | 右移，单词首 |
| `b` | 左移，单词首 |
| `e` | 右移，单词尾 |
| `nw` , `nb` | n表示数字|
|  | **行内,字符** |
| `fx` | 本行，下一个`x`字符|
|  | **行内,一行** |
|`0` or `^` | 行首 |
| `$` | 行尾 |
| `n$` | n行行尾 |
| |  **跨行** |
| `gg` | 文件头|
| `G` | 文件尾 |
| `nG` | 第n行 |
| `:n` | 第n行（编辑模式） |
| | **匹配号** |
| `%` | 找到当前字符的匹配号。匹配符有`[]`,`{}`,`()`| 


## 插入
| 快捷键 | 说明 |
| - | - |
| `esc` > `o` | 向下插入一行 |

## 删除

| 快捷键 | 说明 |
| - | - |
| `esc` > `dd`   | 删除一行 |
| `esc` > `5dd` | 删除光标下面5行 |
| `esc` > `3,5d` | 删除3到5行<br/>1. `.`当前行<br/>2. `$`最后一行<br/>3. `%`所有行|
| `esc` > `%d` | 删除所有行 |
